## What You Actually Need

### Option 1: Replicate (RECOMMENDED ⭐)

**What it is:** Platform that runs AI models for you

**What you get:**
- ✅ Flux model hosted and ready
- ✅ Simple API calls
- ✅ Pay per image ($0.05)
- ✅ No infrastructure setup

**Setup (5 minutes):**
1. Go to https://replicate.com
2. Sign up (free account)
3. Get API token
4. Add to Cloudflare secrets: `REPLICATE_API_TOKEN`
5. Done

**Code:**
```typescript
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

async function generateImage(prompt: string) {
  const output = await replicate.run(
    "black-forest-labs/flux-pro",
    {
      input: {
        prompt: prompt,
        aspect_ratio: "16:9",
        num_outputs: 1,
        guidance: 3.5,
      },
    }
  );
  return output[0]; // Returns image URL
}
```

## Recommended Setup: Replicate + Flux

### Step 1: Create Replicate Account

1. Go to https://replicate.com
2. Click "Sign Up"
3. Use email or GitHub login
4. Verify email

### Step 2: Get API Token

1. Login to Replicate dashboard
2. Click your profile (top right)
3. Select "API tokens"
4. Click "Create token"
5. Name it: `mood-board-flux`
6. Copy the token (looks like: `r8_xxxxxxxxxxxx`)

### Step 3: Add to Cloudflare

**Cloudflare Dashboard → Workers → Settings → Environment Variables**

```
Variable name: REPLICATE_API_TOKEN
Value: r8_xxxxxxxxxxxx (paste your token)
```

**Or in wrangler.toml:**
```toml
[env.production]
vars = { REPLICATE_API_TOKEN = "r8_xxxxxxxxxxxx" }
```

### Step 4: Add to Worker Code

**functions/index.js:**
```javascript
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

export async function generateMockupImage(prompt, aspectRatio = "16:9") {
  try {
    const output = await replicate.run(
      "black-forest-labs/flux-pro",
      {
        input: {
          prompt: prompt,
          aspect_ratio: aspectRatio,
          num_outputs: 1,
          guidance: 3.5,
          output_format: "png",
          output_quality: 80,
        },
      }
    );
    
    return {
      success: true,
      imageUrl: output[0],
      generatedAt: new Date().toISOString(),
    };
  } catch (error) {
    return {
      success: false,
      error: error.message,
    };
  }
}
```

### Step 5: Test It

**Simple test in Node:**
```javascript
const result = await generateMockupImage(
  "Minimalist fitness dashboard hero. Navy background with orange circle. White text."
);
console.log(result.imageUrl); // URL to generated image
```

## Common Issues & Solutions

### Issue: "Authentication failed"
**Solution:** 
- Check `REPLICATE_API_TOKEN` is set in Cloudflare secrets
- Verify token is correct (copy-paste carefully)
- Try regenerating token on Replicate dashboard

### Issue: "Image generation takes >30 seconds"
**Solution:**
- Normal. Flux takes 10-20 seconds
- Show loading spinner: "Generating mockups... (2/6 complete)"
- Consider showing placeholder while generating

### Issue: "Images don't match the prompt"
**Solution:**
- Adjust `guidance` value (default 3.5)
- Higher guidance = more adherent to prompt (3.5-7.5)
- Refine prompt wording
- Allow user to regenerate with tweaked prompt

### Issue: "Cost is higher than expected"
**Solution:**
- Check aspect ratios (all same, or different?)
- Check guidance value (higher = more expensive? Actually no, same price)
- Replicate charges per image, not per parameter
- At $0.05/image, 1,000 projects = $250-300 max
